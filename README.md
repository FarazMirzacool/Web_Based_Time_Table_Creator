📅 Smart Time Table Scheduler - Intelligent Course & Schedule Management System

🎯 Overview
Smart Time Table Scheduler is an  application that can be used for creating and managing academic timetables. It eliminates scheduling conflicts, optimizes resource utilization, and adapts to dynamic changes. Whether you're managing a school, college, or university, this system streamlines the entire scheduling process with intelligent algorithms and an intuitive interface.

✨ Key Features
🧠 Intelligent Scheduling
Conflict-Free Timetables: Automatically detects and prevents scheduling conflicts

Resource Optimization: Optimal allocation of rooms, labs, and equipment

Teacher Preferences: Considers teacher availability, preferences, and workload

Batch Management: Handles multiple batches, sections, and elective groups


👥 User Roles
Administrators: Full system control and configuration

Department Heads: Department-specific schedule management

Teachers: View schedules, request changes, mark availability

Students: Access personal timetables and updates

🛠️ Technology Stack
Frontend
React.js 18 - UI library with hooks and context

React Big Calendar - Calendar visualization

Axios - API integration

Date-fns - Date manipulation

Backend
Node.js - Runtime environment

Express.js - Web framework

MongoDB - Primary database

JWT - Authentication

Socket.io - Real-time notifications


📋 Prerequisites
Ensure you have installed:

Node.js (v16 or higher)

MongoDB (v5.0 or higher)

npm or yarn package manager

Manual Scheduling Complexity

Hours of manual effort every semester

Difficulty accommodating all constraints

Underutilized classrooms and labs

Uneven teacher workload distribution

Inefficient use of time slots

Conflict Management

Teacher double-booking

Classroom clashes

Student group conflicts

Dynamic Changes
